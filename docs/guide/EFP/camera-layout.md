# 机位设计

机位设计是多机位制作的核心环节，合理的机位布局能够确保拍摄内容的完整性和画面的流畅性。

## 机位分类

### 基础机位类型

#### 广角全景机位
- **功能**：客观记录，接近观众视角
- **重要性**：是布置其他机位的基础，其他机位有问题时的保障
- **注意事项**：不应该被其他机位遮挡

#### 长焦机位
- **功能**：拍摄近景/特写，捕捉人眼看不到的细节
- **特点**：高放大倍率，适合拍摄发言人特写
- **要求**：需要高级三脚架确保画面稳定

#### 普通游机
- **特点**：非常灵活，用于补充其他机位的表达
- **镜头选择**：一般使用广角变焦镜头，获得更强造型感
- **注意事项**：
  - 拍摄重要人物反应时不能用广角（避免畸变）
  - 必须配备助理负责顺线、收线、协助转移

### 特殊性质机位

用于实现长线运动镜头（起幅落幅、长时间、完整呈现）：

| 设备类型 | 运动范围 | 特点 | 注意事项 |
|----------|----------|------|----------|
| **摇臂** | 4-40米伸缩 | 配轨道或轮子实现多维度运动 | 需要专业操作员 |
| **斯坦尼康** | 自由移动 | 配广角镜头，画面稳定 | 无线传输可能被干扰 |
| **轨道** | 直线/弧线 | 普通轨道、电控轨道 | 铺设时间较长 |
| **飞猫** | 高空索道 | 2维/3维可升降系统 | 安全要求高 |
| **无人机** | 三维空间 | 实时回传或后期剪辑 | 需要飞行许可 |

## 机位布局原则

::: tip 核心原则
机位布局需要综合考虑内容需求、场地条件、设备能力和团队水平等多个因素。
:::

### 1. 内容导向原则
根据活动内容和场地特点选择合适的设备配置，不同场景有相应的成熟方案可参考。

### 2. 分组管理原则
**相同性质机位成组设置**：
- 将功能相近的机位归为一组
- 方便导播统一调度
- 让画面切换更流畅自然
- 例如：负责拍摄中景的多个机位可成组布置

### 3. 功能最大化原则
**合理配置机位镜头与参数**：
- 挖掘机位多功能性
- 固定机位在常规拍摄外，也可拍摄观众反应
- 一机多用，提高设备利用率

### 4. 技术考量原则
- **光线条件**：考虑自然光和人工照明的影响
- **画面构图**：遵循构图美学原则
- **信号传输**：确保稳定的信号传输路径

### 5. 观众体验原则
从观众视角出发，布局机位时考虑：
- 观众观看习惯
- 视觉感受
- 心理预期

### 6. 团队能力原则
根据摄影师和团队的技术水平合理分配机位任务。

## 机位设计流程

### 第一阶段：前期勘查与规划

**实地勘察要点：**
- 了解场地空间结构、出入口、舞台位置
- 评估光线条件和电源分布
- 确认安全通道和紧急出口

**沟通协调：**
- 与主办方明确拍摄重点、流程和特殊要求
- 了解活动时间安排和人员流动
- 确认拍摄权限和限制区域

**初步设计：**
- 标记各机位大致位置
- 规划拍摄画面和角度
- 制作机位布局图

### 第二阶段：确定设备清单

**设备准备：**
- 根据机位设计方案准备摄像设备、镜头、三脚架等
- 计算线缆长度，预留冗余
- 准备备用设备和应急方案

**设备调试：**
- 检查画面质量、焦距调节、光圈设置
- 测试信号传输稳定性
- 特殊设备（摇臂、无人机）安全检查和测试

### 第三阶段：进场架设

**安装要点：**
- 确保三脚架稳固
- 线缆铺设安全，不影响人员通行
- 做好线缆固定和标识

### 第四阶段：预演与调整

**彩排要求：**
- 至少进行一次完整彩排
- 各机位摄像师按流程拍摄
- 导播观察画面效果和机位配合

**优化调整：**
- 根据预演情况调整机位位置
- 优化拍摄参数设置
- 完善应急预案
