# 收声方案
声音是视频的重要组成部分，在关注画面的同时不能忽视声音的质量。**任何一次现场录制活动都应该考虑声音！**

多机位制作可以选择的收声方案种类较多，可以根据活动类型、经费、需求确定需要在收音方面的工作量。按复杂程度依次排序可以粗略分为：相机麦克风直接录制、使用手机专门采集需要的声音、从调音台录制、专门架设麦克风录制。
